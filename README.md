# Digital Board Game

## Overview
This is a digital board game implemented using HTML, CSS, and JavaScript.  
The game includes multiple paths and mini-games, providing an engaging and interactive experience.  

> ⚠️ **Note:** This game is only optimized for laptops and desktop computers. Mobile devices may not display the game correctly.

## Features
- Dynamic board with multiple paths.
- Variety of mini-games with unique mechanics.
- Player movement and game logic implemented.
- Interactive UI with visual feedback.
- Easily expandable to add new paths or mini-games.
